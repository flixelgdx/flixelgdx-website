This project is configured for **GraalVM Community**.
Downloads: https://www.graalvm.org/downloads/

Quick-start by OS:
  - Easiest path: SDKMAN. Then `sdk install java 17-graal`.
  - Windows : unzip into C:\graalvm, add the `bin` folder to PATH, set JAVA_HOME.
  - macOS / Linux: SDKMAN handles PATH automatically.

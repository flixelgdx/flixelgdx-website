This project is configured for **Amazon Corretto**.
Downloads: https://docs.aws.amazon.com/corretto/latest/corretto-17-ug/downloads-list.html

Quick-start by OS:
  - Windows : run the .msi installer.
  - macOS   : brew install --cask corretto@17
  - Linux   : configure the Amazon apt/yum repo + install `java-17-amazon-corretto-jdk`.

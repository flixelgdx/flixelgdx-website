# FlixelGDX API Reference

This is the live Java API reference for **FlixelGDX**. It is generated
from the framework's sources by [Dokka](https://kotlinlang.org/docs/dokka-introduction.html)
with the **kotlin-as-java** plugin (so every signature reads as native
Java) and re-shaped for the website by `scripts/postprocess_api.py`.

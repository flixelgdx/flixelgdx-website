# FlixelGDX API Reference

This is the live Java API reference for the FlixelGDX framework. All pages you see here are auto-generated from the
framework's Javadoc comments in its source code, so they are always up to date with the latest release. They are parsed
and translated into Markdown files using the frameworks own custom tool [DocletMD](https://github.com/flixelgdx/DocletMD).

# FlixelGDX API Reference

This is the live API reference for the **FlixelGDX** framework. It is
generated automatically from the Java sources of
[`flixelgdx/flixelgdx`](https://github.com/flixelgdx/flixelgdx) every
time the `master` branch is updated.

Pick a **package** from the left sidebar to start browsing — each
package collapses to its classes, and each class page lists the
properties, methods and constructors as their own sections (so you can
deep-link to a specific member and Ctrl-F across them quickly).

If you spot something wrong in the API docs, please
[file an issue](https://github.com/flixelgdx/flixelgdx/issues) against
the **framework** repository — the API content is regenerated from
there on every push.
